## 环境

- Nodejs [lts](https://nodejs.org/en/download), 👉[教程戳这里](https://www.liaoxuefeng.com/wiki/1022910821149312/1023025597810528)

## 安装依赖

```bash
npm i
```

## 配置

打开 .env 文件填写私钥

## 运行

```bash
node main.js
```
